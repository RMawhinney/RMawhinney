LECTURE: BOOTSTRAP

Bootstrap is framework to build from (mobile first and responsive design, battle tested, modern web tech)
Framework: Way to organize code following structure that enables abilities (built in CSS, JavaScript
Bootstrap: excellent for small, quick projects, prototyping and scaffolding sites, and great for larger projects (if so; customize your install)

Basic/Advanced installation
Required jQuery

Creating basic template using CDN
Download Bootstrap 3
BS comes with a 12-column grid accessed through CSS classes

Three key concepts: containers, rows, columns
 Container: controls layout and adds padding
 2 types: fixed and fluid. Fluid adjusts to browser size
   Fixed-width comes with 15px padding on each side and adjusts to media query breakpoints
 Rows: horizontal groups of columns, place within container. Should always include columns, and gets rid of container padding
 Columns: 30px gutters(15px ea side)
 Use col-SIZE-SPAN
   Size: when it stacks
   SPAN: how many columns
   >12 columns wrap
