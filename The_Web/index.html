Project 0 The Web

SUBJECT: The Web

Once one has an understanding of what goes into creating a website, how to develop it, as well as knowing what it is you're trying to sell or promote, it's important to be versed in the environment in which you will be doing the majority of your business. The web is a very vast resource with many rules and laws not immediately apparent to the layman, but still necessary to be accessible and intuitive. Understanding what goes into making the web accessible (or perhaps not as accessible) is vital to making the most of what the World Wide Web already has available. If one were so inclined, the most valuable information on this site would be under this section, because knowing the situation and reasons for limitation to a tool is almost more important then how to use a tool itself. This section covers Web 2.0, Web Tech, Sites vs Apps, the Darkweb, UI / UX, and the most recent Chrome Dev Summit.
